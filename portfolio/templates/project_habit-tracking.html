This project is a Flask application that allows users to track their habits. It includes routes for the home page, adding a habit, and marking a habit as complete.

The home page shows a list of habits and allows the user to select a date to view habits for that day. The add a habit page allows the user to add a new habit and the complete page allows the user to mark a habit as complete for a specific date.

The code uses the datetime, uuid, and Flask libraries and uses a MongoDB database to store information about habits and completions. The context processor function date_range is used to calculate the date range.

Technologies used

  • Python
  • Flask
  • HTML
  • CSS
